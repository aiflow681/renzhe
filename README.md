【游戏名称】
忍者突袭

【游戏介绍】
这是一款H5原生开发的动作防御类小游戏。游戏中，你扮演一名忍者，面对从上方不断来袭的敌人武士。你需要通过投掷飞镖消灭敌人，或者使用闪避技能躲过攻击，保护自己的防线！

【玩法说明】
游戏中，敌人武士会从屏幕上方的4条泳道中随机出现，不断向下移动攻击你。

障碍：
1. 敌人武士会从上方不断来袭，速度会随着得分增加而加快。
2. 如果敌人碰到你或到达底部防线，你会失去一颗生命值。
3. 你只有3颗生命值，失去所有生命值游戏就会结束。

操作：
点击屏幕右侧（或按右方向键/D键）- 投掷飞镖攻击敌人
点击屏幕左侧（或按左方向键/A键）- 闪避攻击，短时间内无敌

目标：
消灭尽可能多的敌人，获得更高的分数。每消灭一个敌人得10分！

Tips：飞镖具有自动追踪功能，会向最近的敌人飞去。
闪避时你会进入无敌状态，敌人无法伤害你。合理使用闪避和攻击，挑战最高分记录！

【技术介绍】
HTML5原生开发，主要语言是JavaScript，使用了Canvas 2D渲染技术和SVG矢量图形，纯原生实现无需任何框架。
