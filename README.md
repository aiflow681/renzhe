忍者突袭 - 游戏说明

【任务简介】
你是一名在执行秘密任务的忍者。敌对武士集团从天而降，企图阻止你。你需要使用手里的飞镖击退敌人，利用灵巧的身法躲避攻击！

【操作说明】
屏幕下方有两个控制区域：

1. 攻击 (Attack) - 右侧按钮
   - 点击发射飞镖。
   - 飞镖会自动追踪距离最近的敌人。
   - 击中敌人可得分。

2. 躲避 (Dodge) - 左侧按钮
   - 点击进行快速闪避移动。
   - 闪避期间可以穿过敌人而不受伤害（仅在紧急关头使用！）。
   - 注意：如果敌人没有被击杀并落到屏幕底部，你依然会扣除生命值。

【游戏规则】
1. 生命值：
   - 玩家拥有 3 颗红心。
   - 如果敌人直接撞击到未处于“闪避”状态的忍者，扣除 1 颗红心。
   - 如果敌人成功突破防线到达屏幕底部，扣除 1 颗红心。

2. 难度升级：
   - 分数越高，敌人下落的速度越快，生成的频率也越高。

3. 目标：
   - 尽可能久地存活，击杀更多敌人，挑战最高分！
